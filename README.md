# Comick Recommendations

A Next.js application that uses the Comick.io API to generate recommendations for a user based on the recommendations of the comics they read.

## Getting Started

### Prerequisites

- Node.js 18.0 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/GooglyBlox/comick-recommendations.git
cd comick-recommendations
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
